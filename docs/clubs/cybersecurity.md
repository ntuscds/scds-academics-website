# Cybersecurity Club

Welcome to the Cybersecurity Club, a specialized community within SCDS focused on all aspects of information security and privacy.

## About the Cybersecurity Club

The Cybersecurity Club brings together students interested in network security, cryptography, ethical hacking, digital forensics, and security policy. We provide a collaborative environment to learn about security concepts, practice defensive and offensive techniques, and develop security-focused projects.
