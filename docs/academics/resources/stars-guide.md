# STARS Guide

The Student Automated Registration System (STARS) is used for course registration at NUS. This guide will help you navigate the system effectively.

## Understanding STARS

STARS is a bidding system where students use points to "bid" for modules they want to take. The system works in rounds, with different modules available at different times.

### Key Components

- **Student Access Period**: The timeframe when you can access STARS.
- **Bidding Points**: You get a set number of points each semester to bid for modules.
- **Bidding Rounds**: Multiple rounds where you can place and adjust bids.
- **Module Quotas**: Limited slots available for each module.

## Preparing for Module Registration

### Before Bidding Starts

1. **Plan Your Modules**: Decide which modules you need and want to take.
2. **Check Prerequisites**: Ensure you meet all prerequisites for your desired modules.
3. **Research Historical Point Data**: Look at previous semesters' successful bid points on NUSMods.
4. **Prepare Backup Plans**: Have alternative modules in mind in case you don't secure your first choices.

### Bidding Strategy

- **Prioritize Core Modules**: Allocate more points to core modules required for graduation.
- **Consider Popularity**: More popular modules typically require higher bid points.
- **Use Points Wisely**: Don't overbid unnecessarily but ensure you have enough points for important modules.
- **Balance Your Schedule**: Consider timing conflicts and workload distribution.

## Navigating the STARS Interface

### Logging In

1. Access STARS through the NUS student portal.
2. Use your NUSNET ID and password.
3. Check your access time carefully.

### Placing Bids

1. Select the "Add Module" function.
2. Enter the module code and number of points you wish to bid.
3. Confirm your bid and keep the confirmation number.

### Checking Results

1. After each bidding round, check if your bids were successful.
2. If unsuccessful, adjust your strategy for subsequent rounds.
3. Use the "View Modules" function to see your registered modules.

## Troubleshooting Common Issues

### Unable to Bid for a Module

Possible reasons:
- Prerequisites not met
- Time conflict with another module
- Module quota filled
- Insufficient bid points

### Withdrawing from Modules

If you need to drop a module:
1. Use the "Drop Module" function.
2. Be aware of the refund policy for points.
3. Check academic calendar for last day to drop modules without penalty.
